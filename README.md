# Information-Gathering-OSINT-Tool

A Python-based reconnaissance tool developed to collect IP address details and geolocation information for early-stage security assessments.

## Features

- IP Address Lookup
- Geolocation Information Retrieval
- Network Intelligence Gathering
- Structured Output Generation

## Technologies Used

- Python
- Socket Programming
- ipinfo.io API
- OSINT Techniques

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python info_tool.py
```
